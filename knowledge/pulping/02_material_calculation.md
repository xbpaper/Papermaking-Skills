# 物料计算

## 1. 原料计算

### 1.1 木材原料计算

**计算公式**：
```
绝干木材量 = 湿木材量 × (1 - 含水率%)
```

**实例**：
- 湿木材量：1000kg，含水率：20%
- 绝干木材量：1000×(1-20%) = 800kg

### 1.2 草类原料计算

**计算公式**：
```
绝干草类量 = 湿草类量 × (1 - 含水率%)
```

**实例**：
- 湿稻草量：1000kg，含水率：15%
- 绝干稻草量：1000×(1-15%) = 850kg

### 1.3 废纸原料计算

**计算公式**：
```
绝干废纸量 = 湿废纸量 × (1 - 含水率%)
```

**实例**：
- 湿废纸量：1000kg，含水率：10%
- 绝干废纸量：1000×(1-10%) = 900kg

## 2. 化学品计算

### 2.1 蒸煮化学品计算

#### 2.1.1 硫酸盐法

**计算公式**：
```
活性碱用量（以Na₂O计）= 绝干原料量 × 用碱量（%）
NaOH用量 = 活性碱用量 × 0.77
Na₂S用量 = 活性碱用量 × 0.37
```

**实例**：
- 绝干木材量：1000kg
- 用碱量：18%（以Na₂O计）
- 活性碱用量：1000×18% = 180kg
- NaOH用量：180×0.77 = 138.6kg
- Na₂S用量：180×0.37 = 66.6kg

#### 2.1.2 亚硫酸盐法

**计算公式**：
```
SO₂用量 = 绝干原料量 × SO₂用量（%）
```

**实例**：
- 绝干木材量：1000kg
- SO₂用量：10%
- SO₂用量：1000×10% = 100kg

### 2.2 漂白化学品计算

#### 2.2.1 氯气漂白

**计算公式**：
```
Cl₂用量 = 绝干浆量 × Cl₂用量（%）
```

**实例**：
- 绝干浆量：1000kg
- Cl₂用量：3%
- Cl₂用量：1000×3% = 30kg

#### 2.2.2 二氧化氯漂白

**计算公式**：
```
ClO₂用量 = 绝干浆量 × ClO₂用量（%）
```

**实例**：
- 绝干浆量：1000kg
- ClO₂用量：0.8%
- ClO₂用量：1000×0.8% = 8kg

#### 2.2.3 过氧化氢漂白

**计算公式**：
```
H₂O₂用量 = 绝干浆量 × H₂O₂用量（%）
```

**实例**：
- 绝干浆量：1000kg
- H₂O₂用量：2%
- H₂O₂用量：1000×2% = 20kg

## 3. 得率计算

### 3.1 蒸煮得率

**计算公式**：
```
蒸煮得率(%) = (粗浆绝干质量 / 绝干原料质量) × 100%
```

**实例**：
- 绝干木材量：1000kg
- 粗浆绝干质量：450kg
- 蒸煮得率：(450/1000)×100% = 45%

### 3.2 总得率

**计算公式**：
```
总得率(%) = (成品浆绝干质量 / 绝干原料质量) × 100%
```

**实例**：
- 绝干木材量：1000kg
- 成品浆绝干质量：420kg
- 总得率：(420/1000)×100% = 42%

## 4. 能耗计算

### 4.1 蒸汽消耗

**计算公式**：
```
蒸汽消耗量 = 绝干原料量 × 单位蒸汽消耗
```

**实例**：
- 绝干木材量：1000kg
- 单位蒸汽消耗：2.5吨/吨绝干原料
- 蒸汽消耗量：1000×2.5 = 2500kg = 2.5吨

### 4.2 电力消耗

**计算公式**：
```
电力消耗量 = 绝干原料量 × 单位电力消耗
```

**实例**：
- 绝干木材量：1000kg
- 单位电力消耗：500kWh/吨绝干原料
- 电力消耗量：1000×0.5 = 500kWh

## 5. 废水计算

### 5.1 黑液量计算

**计算公式**：
```
黑液量 = 绝干原料量 × 单位黑液产生量
```

**实例**：
- 绝干木材量：1000kg
- 单位黑液产生量：15m³/吨绝干原料
- 黑液量：1000×15 = 15m³

### 5.2 废水污染物计算

**计算公式**：
```
COD排放量 = 废水排放量 × COD浓度
BOD排放量 = 废水排放量 × BOD浓度
SS排放量 = 废水排放量 × SS浓度
```

**实例**：
- 废水排放量：15m³
- COD浓度：15000mg/L = 15g/L
- COD排放量：15×15 = 225kg

## 6. 成本计算

### 6.1 原料成本

**计算公式**：
```
原料成本 = 绝干原料量 × 原料单价
```

**实例**：
- 绝干木材量：1000kg = 1吨
- 原料单价：800元/吨绝干
- 原料成本：1×800 = 800元

### 6.2 化学品成本

**计算公式**：
```
化学品成本 = Σ(化学品用量 × 化学品单价)
```

**实例**：
- NaOH用量：138.6kg，单价：3元/kg
- Na₂S用量：66.6kg，单价：4元/kg
- 化学品成本：138.6×3 + 66.6×4 = 415.8 + 266.4 = 682.2元

### 6.3 能源成本

**计算公式**：
```
能源成本 = 蒸汽消耗量 × 蒸汽单价 + 电力消耗量 × 电力单价
```

**实例**：
- 蒸汽消耗量：2.5吨，单价：200元/吨
- 电力消耗量：500kWh，单价：0.6元/kWh
- 能源成本：2.5×200 + 500×0.6 = 500 + 300 = 800元

### 6.4 总成本

**计算公式**：
```
总成本 = 原料成本 + 化学品成本 + 能源成本 + 人工成本 + 设备折旧 + 其他成本
```

**实例**：
- 原料成本：800元
- 化学品成本：682.2元
- 能源成本：800元
- 人工成本：300元
- 设备折旧：200元
- 其他成本：100元
- 总成本：800 + 682.2 + 800 + 300 + 200 + 100 = 2882.2元

### 6.5 单位成本

**计算公式**：
```
单位成本 = 总成本 / 成品浆绝干质量
```

**实例**：
- 总成本：2882.2元
- 成品浆绝干质量：420kg
- 单位成本：2882.2 / 420 ≈ 6.86元/kg

## 7. 物料平衡计算实例

**已知条件**：
- 绝干木材量：1000kg
- 蒸煮得率：45%
- 筛选得率：95%
- 漂白得率：98%
- 用碱量：18%（以Na₂O计）
- 单位蒸汽消耗：2.5吨/吨绝干原料
- 单位电力消耗：500kWh/吨绝干原料

**计算过程**：

1. **各工序物料量**：
   - 粗浆绝干质量：1000×45% = 450kg
   - 筛选后浆绝干质量：450×95% = 427.5kg
   - 成品浆绝干质量：427.5×98% = 418.95kg

2. **化学品用量**：
   - 活性碱用量：1000×18% = 180kg
   - NaOH用量：180×0.77 = 138.6kg
   - Na₂S用量：180×0.37 = 66.6kg

3. **能源消耗**：
   - 蒸汽消耗量：1000×2.5 = 2500kg = 2.5吨
   - 电力消耗量：1000×0.5 = 500kWh

4. **成本计算**：
   - 原料成本：1000×0.8 = 800元（假设木材单价800元/吨）
   - 化学品成本：138.6×3 + 66.6×4 = 682.2元
   - 能源成本：2.5×200 + 500×0.6 = 800元
   - 总成本：800 + 682.2 + 800 + 300 + 200 + 100 = 2882.2元
   - 单位成本：2882.2 / 418.95 ≈ 6.88元/kg

**结论**：
- 每处理1吨绝干木材，可生产418.95kg绝干成品浆
- 单位成品浆成本约为6.88元/kg
