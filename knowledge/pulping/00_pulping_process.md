# 制浆工艺详解

## 1. 化学制浆

### 1.1 硫酸盐法（Kraft法）

**原理**：使用氢氧化钠和硫化钠的混合液作为蒸煮液，在高温高压下分解木素，释放纤维素纤维。

**工艺流程**：

1. 备料：原木剥皮、切片
2. 蒸煮：在蒸煮器中加入木片和蒸煮液，在170-180℃、1.0-1.5MPa下蒸煮2-4小时
3. 放料与喷放：将蒸煮后的浆料喷放到喷放仓
4. 洗涤：用黑液提取机提取黑液，浆料经多段洗涤
5. 筛选：去除未煮透的木片和杂质
6. 漂白：根据需要进行多段漂白

**特点**：

- 得率：45-55%
- 优点：浆料强度高，适应性广，可处理各种木材
- 缺点：黑液处理复杂，投资大

**适用范围**：大规模制浆厂，生产高强度纸种

### 1.2 亚硫酸盐法

**原理**：使用亚硫酸和亚硫酸盐的混合液作为蒸煮液，在酸性条件下分解木素。

**工艺流程**：

1. 备料：原木剥皮、切片
2. 蒸煮：在蒸煮器中加入木片和蒸煮液，在130-160℃下蒸煮8-12小时
3. 放料与喷放：将蒸煮后的浆料喷放到喷放仓
4. 洗涤：提取红液，浆料经多段洗涤
5. 筛选：去除未煮透的木片和杂质
6. 漂白：通常需要较少的漂白剂

**特点**：

- 得率：50-65%
- 优点：浆料颜色浅，易漂白，得率较高
- 缺点：对原料要求高，设备腐蚀严重

**适用范围**：针叶木原料，生产高白度纸种

## 2. 机械制浆

### 2.1 磨石磨木浆（SGW）

**原理**：将原木直接在磨石上磨碎，通过机械力分离纤维。

**工艺流程**：

1. 原木预处理：剥皮、截段
2. 磨浆：原木在旋转的磨石上摩擦，加入大量水
3. 筛选：去除粗大纤维和杂质
4. 浓缩：将浆料浓缩至适当浓度

**特点**：

- 得率：90-95%
- 优点：得率高，成本低
- 缺点：浆料强度低，易返黄

**适用范围**：新闻纸、卫生纸等低强度纸种

### 2.2 热磨机械浆（TMP）

**原理**：将木片在预热器中预热软化，然后在盘磨机中磨浆。

**工艺流程**：

1. 备料：原木剥皮、切片
2. 预热：木片在预热器中加热至100-130℃
3. 磨浆：预热后的木片在盘磨机中磨浆
4. 筛选：去除粗大纤维和杂质
5. 浓缩：将浆料浓缩至适当浓度

**特点**：

- 得率：85-90%
- 优点：浆料强度高于SGW，得率高
- 缺点：能耗较高

**适用范围**：新闻纸、印刷纸等

## 3. 半化学制浆

### 3.1 中性亚硫酸盐半化学浆（NSSC）

**原理**：使用中性亚硫酸盐溶液轻度蒸煮，然后通过机械力分离纤维。

**工艺流程**：

1. 备料：原木剥皮、切片
2. 蒸煮：在中性亚硫酸盐溶液中蒸煮，温度160-180℃，时间1-2小时
3. 磨浆：蒸煮后的木片在盘磨机中磨浆
4. 筛选：去除杂质
5. 浓缩：将浆料浓缩至适当浓度

**特点**：

- 得率：65-80%
- 优点：得率高，强度适中
- 缺点：对原料要求较高

**适用范围**：瓦楞纸、箱板纸等包装纸

### 3.2 化学机械浆（CMP）

**原理**：先用少量化学药品处理，然后进行机械磨浆。

**工艺流程**：

1. 备料：原木剥皮、切片
2. 化学预处理：用氢氧化钠或亚硫酸盐溶液处理
3. 磨浆：在盘磨机中磨浆
4. 筛选：去除杂质
5. 浓缩：将浆料浓缩至适当浓度

**特点**：

- 得率：75-85%
- 优点：强度高于机械浆，得率高于化学浆
- 缺点：能耗较高

**适用范围**：印刷纸、包装纸等

## 4. 废纸制浆

**原理**：通过机械和化学方法处理废纸，分离纤维。

**工艺流程**：

1. 废纸分类：按品质和用途分类
2. 碎解：在碎浆机中加水碎解废纸
3. 筛选：去除杂质（塑料、金属等）
4. 除渣：去除重杂质
5. 浮选：去除油墨和其他轻杂质
6. 洗涤：进一步去除杂质
7. 浓缩：将浆料浓缩至适当浓度

**特点**：

- 得率：取决于废纸品质，一般70-90%
- 优点：原料成本低，环保
- 缺点：浆料质量波动大，强度较低

**适用范围**：箱板纸、瓦楞纸、卫生纸等

## 5. 非木材制浆

### 5.1 草类制浆

**原理**：使用化学方法处理草类原料（如稻草、麦草、芦苇等）。

**工艺流程**：

1. 备料：切断、除尘
2. 蒸煮：使用氢氧化钠和硫化钠溶液蒸煮
3. 洗涤：提取黑液
4. 筛选：去除杂质
5. 漂白：根据需要进行漂白

**特点**：

- 得率：40-50%
- 优点：原料来源广泛
- 缺点：黑液处理困难，浆料强度较低

**适用范围**：文化用纸、包装纸等

### 5.2 竹浆

**原理**：使用化学方法处理竹材。

**工艺流程**：

1. 备料：切断、除尘
2. 蒸煮：使用硫酸盐法或亚硫酸盐法蒸煮
3. 洗涤：提取黑液
4. 筛选：去除杂质
5. 漂白：根据需要进行漂白

**特点**：

- 得率：45-55%
- 优点：浆料强度高，纤维细长
- 缺点：原料收集成本较高

**适用范围**：高级文化用纸、特种纸等

## 6. 工艺选择指南

### 6.1 根据原料选择

- 针叶木：硫酸盐法、亚硫酸盐法、TMP
- 阔叶木：硫酸盐法、NSSC、CMP
- 废纸：废纸制浆工艺
- 草类：硫酸盐法、烧碱法
- 竹子：硫酸盐法

### 6.2 根据产品选择

- 高级文化用纸：硫酸盐法、亚硫酸盐法
- 新闻纸：TMP、SGW
- 包装纸：NSSC、废纸制浆
- 卫生纸：机械浆、废纸制浆

### 6.3 根据规模选择

- 大型厂（>1000吨/日）：硫酸盐法
- 中型厂（300-1000吨/日）：硫酸盐法、NSSC
- 小型厂（<300吨/日）：废纸制浆、草类制浆
