# 浆水平衡计算

## 1. 基本概念

### 1.1 浓度

**定义**：浆料中绝干纤维质量与浆料总质量的比值，通常用百分比表示。

**计算公式**：
```
浓度(%) = (绝干纤维质量 / 浆料总质量) × 100%
```

### 1.2 绝干纤维质量

**定义**：浆料中不含水分的纤维质量。

**计算公式**：
```
绝干纤维质量 = 浆料质量 × 浓度%
```

### 1.3 浆水平衡

**定义**：在制浆过程中，输入的纤维和水的总量等于输出的纤维和水的总量。

**基本公式**：
- 纤维平衡：输入纤维量 = 输出纤维量
- 水平衡：输入水量 = 输出水量

## 2. 制浆过程浆水平衡计算

### 2.1 蒸煮过程浆水平衡

**输入**：
- 木片（绝干）：M₁
- 蒸煮液：V₁（体积），浓度：C₁（%）
- 补充水：V₂

**输出**：
- 粗浆：M₂（绝干），浓度：C₂（%）
- 黑液：V₃（体积），浓度：C₃（%）

**平衡方程**：
1. 纤维平衡：M₁ = M₂
2. 水平衡：V₁×(1-C₁/100) + V₂ = M₂×(100/C₂ - 1) + V₃×(1-C₃/100)

**实例**：
- 输入：木片（绝干）1000kg，蒸煮液10m³（浓度15%），补充水5m³
- 输出：粗浆得率45%，浓度12%
- 计算：
  - 粗浆绝干质量：1000×45% = 450kg
  - 粗浆总质量：450 / 12% = 3750kg
  - 粗浆含水量：3750 - 450 = 3300kg
  - 蒸煮液含水量：10×1000×(1-15%) = 8500kg
  - 补充水：5×1000 = 5000kg
  - 总输入水量：8500 + 5000 = 13500kg
  - 黑液含水量：13500 - 3300 = 10200kg
  - 黑液中固形物：1000 - 450 + 10×1000×15% = 550 + 1500 = 2050kg
  - 黑液浓度：2050 / (10200 + 2050) × 100% ≈ 16.7%
  - 黑液体积：(10200 + 2050) / 1000 ≈ 12.25m³

### 2.2 洗涤过程浆水平衡

**输入**：
- 粗浆：M₁（绝干），浓度：C₁（%）
- 洗涤水：V₂

**输出**：
- 洗净浆：M₁（绝干），浓度：C₂（%）
- 稀黑液：V₃（体积），浓度：C₃（%）

**平衡方程**：
1. 纤维平衡：M₁ = M₁
2. 水平衡：M₁×(100/C₁ - 1) + V₂ = M₁×(100/C₂ - 1) + V₃×(1-C₃/100)

**实例**：
- 输入：粗浆（绝干）450kg，浓度12%，洗涤水8m³
- 输出：洗净浆浓度10%
- 计算：
  - 粗浆总质量：450 / 12% = 3750kg
  - 粗浆含水量：3750 - 450 = 3300kg
  - 洗净浆总质量：450 / 10% = 4500kg
  - 洗净浆含水量：4500 - 450 = 4050kg
  - 洗涤水：8×1000 = 8000kg
  - 总输入水量：3300 + 8000 = 11300kg
  - 稀黑液含水量：11300 - 4050 = 7250kg
  - 稀黑液浓度：假设为5%
  - 稀黑液总质量：7250 / (1-5%) ≈ 7631.58kg
  - 稀黑液体积：7631.58 / 1000 ≈ 7.63m³

### 2.3 筛选过程浆水平衡

**输入**：
- 进浆：M₁（绝干），浓度：C₁（%）

**输出**：
- 良浆：M₂（绝干），浓度：C₂（%）
- 尾浆：M₃（绝干），浓度：C₃（%）

**平衡方程**：
1. 纤维平衡：M₁ = M₂ + M₃
2. 水平衡：M₁×(100/C₁ - 1) = M₂×(100/C₂ - 1) + M₃×(100/C₃ - 1)

**实例**：
- 输入：进浆（绝干）450kg，浓度10%
- 输出：良浆得率95%，浓度12%，尾浆浓度8%
- 计算：
  - 良浆绝干质量：450×95% = 427.5kg
  - 尾浆绝干质量：450 - 427.5 = 22.5kg
  - 进浆总质量：450 / 10% = 4500kg
  - 进浆含水量：4500 - 450 = 4050kg
  - 良浆总质量：427.5 / 12% = 3562.5kg
  - 良浆含水量：3562.5 - 427.5 = 3135kg
  - 尾浆总质量：22.5 / 8% = 281.25kg
  - 尾浆含水量：281.25 - 22.5 = 258.75kg
  - 验证水平衡：3135 + 258.75 = 3393.75kg（与进浆含水量4050kg不符，说明需要补充水或调整浓度）

## 3. 连续制浆系统浆水平衡

### 3.1 系统组成

- 备料系统
- 蒸煮系统
- 洗涤系统
- 筛选系统
- 漂白系统

### 3.2 整体平衡计算

**输入**：
- 原料（绝干）：M
- 工艺水：V₁
- 化学药品溶液：V₂（浓度C₂）

**输出**：
- 成品浆（绝干）：M×得率
- 废水：V₃（浓度C₃）
- 废气：忽略不计

**平衡方程**：
1. 纤维平衡：M = 成品浆绝干质量
2. 水平衡：V₁ + V₂×(1-C₂/100) = 成品浆含水量 + V₃×(1-C₃/100)
3. 化学品平衡：V₂×C₂/100 = 成品浆中化学品残留 + V₃×C₃/100

### 3.3 计算步骤

1. 确定原料量和得率
2. 计算各工序的浓度和流量
3. 建立各工序的浆水平衡方程
4. 联立求解各未知数
5. 验证计算结果

## 4. 浆水平衡计算的应用

### 4.1 工艺优化

通过浆水平衡计算，可以：
- 优化洗涤用水量，减少水消耗
- 优化蒸煮液用量，提高化学品利用率
- 优化筛选系统，提高良浆得率

### 4.2 设备选型

根据浆水平衡计算结果，可以：
- 确定各设备的处理能力
- 选择合适的泵、阀门和管道
- 设计合理的储罐容量

### 4.3 成本控制

通过浆水平衡计算，可以：
- 精确计算原料和化学品消耗
- 预测废水处理成本
- 优化能源消耗

## 5. 实例：硫酸盐法制浆浆水平衡计算

**已知条件**：
- 木片处理量：1000kg绝干/小时
- 蒸煮得率：45%
- 粗浆浓度：12%
- 洗涤水用量：8m³/吨绝干浆
- 洗净浆浓度：10%
- 筛选良浆得率：95%
- 良浆浓度：12%

**计算过程**：

1. **蒸煮过程**：
   - 粗浆绝干质量：1000×45% = 450kg
   - 粗浆总质量：450 / 12% = 3750kg
   - 粗浆含水量：3750 - 450 = 3300kg

2. **洗涤过程**：
   - 洗涤水用量：8×0.45 = 3.6m³ = 3600kg
   - 洗净浆总质量：450 / 10% = 4500kg
   - 洗净浆含水量：4500 - 450 = 4050kg
   - 稀黑液含水量：3300 + 3600 - 4050 = 2850kg

3. **筛选过程**：
   - 良浆绝干质量：450×95% = 427.5kg
   - 尾浆绝干质量：450 - 427.5 = 22.5kg
   - 良浆总质量：427.5 / 12% = 3562.5kg
   - 良浆含水量：3562.5 - 427.5 = 3135kg
   - 尾浆总质量：22.5 / 8% = 281.25kg（假设尾浆浓度8%）
   - 尾浆含水量：281.25 - 22.5 = 258.75kg

4. **总水平衡**：
   - 输入水量：3300（粗浆含水） + 3600（洗涤水） = 6900kg
   - 输出水量：3135（良浆含水） + 258.75（尾浆含水） + 2850（稀黑液含水） = 6243.75kg
   - 差异：6900 - 6243.75 = 656.25kg（可能是蒸发损失或其他损耗）

**结论**：
- 每小时生产良浆427.5kg绝干
- 消耗洗涤水3.6m³
- 产生稀黑液约2.85m³
- 产生尾浆281.25kg
