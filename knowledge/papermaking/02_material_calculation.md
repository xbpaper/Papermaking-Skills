# 物料计算

## 1. 纸浆计算

### 1.1 纸浆用量计算

**计算公式**：
```
绝干纸浆用量 = 纸产量 × (1 - 成品纸水分%)
```

**实例**：
- 纸产量：100吨/日
- 成品纸水分：8%
- 绝干纸浆用量：100×(1-8%) = 92吨/日

### 1.2 纸浆浓度计算

**计算公式**：
```
纸浆浓度(%) = (绝干纸浆质量 / 纸浆总质量) × 100%
```

**实例**：
- 绝干纸浆质量：92吨
- 纸浆总质量：1000吨
- 纸浆浓度：(92/1000)×100% = 9.2%

### 1.3 纸浆配比计算

**计算公式**：
```
各浆种用量 = 绝干纸浆总用量 × 配比%
```

**实例**：
- 绝干纸浆总用量：92吨/日
- 木浆配比：70%
- 废纸浆配比：30%
- 木浆用量：92×70% = 64.4吨/日
- 废纸浆用量：92×30% = 27.6吨/日

## 4. 能源计算

### 4.1 蒸汽消耗

**计算公式**：
```
蒸汽消耗量 = 纸产量 × 单位蒸汽消耗
```

**实例**：
- 纸产量：100吨/日
- 单位蒸汽消耗：2.0吨/吨纸
- 蒸汽消耗量：100×2.0 = 200吨/日

### 4.2 电力消耗

**计算公式**：
```
电力消耗量 = 纸产量 × 单位电力消耗
```

**实例**：
- 纸产量：100吨/日
- 单位电力消耗：600kWh/吨纸
- 电力消耗量：100×600 = 60000kWh/日

### 4.3 燃料消耗

**计算公式**：
```
燃料消耗量 = 纸产量 × 单位燃料消耗
```

**实例**：
- 纸产量：100吨/日
- 单位燃料消耗：0.2吨标准煤/吨纸
- 燃料消耗量：100×0.2 = 20吨标准煤/日

## 5. 成本计算

### 5.1 原料成本

**计算公式**：
```
原料成本 = Σ(原料用量 × 原料单价)
```

**实例**：
- 木浆用量：64.4吨/日，单价：6000元/吨
- 废纸浆用量：27.6吨/日，单价：3000元/吨
- 原料成本：64.4×6000 + 27.6×3000 = 386400 + 82800 = 469200元/日

### 5.2 添加剂成本

**计算公式**：
```
添加剂成本 = Σ(添加剂用量 × 添加剂单价)
```

**实例**：
- 干强剂用量：0.92吨/日，单价：10000元/吨
- 湿强剂用量：0.46吨/日，单价：20000元/吨
- 施胶剂用量：0.736吨/日，单价：15000元/吨
- 填料用量：18.4吨/日，单价：500元/吨
- 添加剂成本：0.92×10000 + 0.46×20000 + 0.736×15000 + 18.4×500 = 9200 + 9200 + 11040 + 9200 = 38640元/日

### 5.3 化学品成本

**计算公式**：
```
化学品成本 = Σ(化学品用量 × 化学品单价)
```

**实例**：
- 硫酸铝用量：1.84吨/日，单价：1000元/吨
- 烧碱用量：0.46吨/日，单价：3000元/吨
- 化学品成本：1.84×1000 + 0.46×3000 = 1840 + 1380 = 3220元/日

### 5.4 能源成本

**计算公式**：
```
能源成本 = 蒸汽消耗量 × 蒸汽单价 + 电力消耗量 × 电力单价 + 燃料消耗量 × 燃料单价
```

**实例**：
- 蒸汽消耗量：200吨/日，单价：200元/吨
- 电力消耗量：60000kWh/日，单价：0.6元/kWh
- 燃料消耗量：20吨标准煤/日，单价：800元/吨
- 能源成本：200×200 + 60000×0.6 + 20×800 = 40000 + 36000 + 16000 = 92000元/日

### 5.5 总成本

**计算公式**：
```
总成本 = 原料成本 + 添加剂成本 + 化学品成本 + 能源成本 + 人工成本 + 设备折旧 + 其他成本
```

**实例**：
- 原料成本：469200元/日
- 添加剂成本：38640元/日
- 化学品成本：3220元/日
- 能源成本：92000元/日
- 人工成本：30000元/日
- 设备折旧：20000元/日
- 其他成本：10000元/日
- 总成本：469200 + 38640 + 3220 + 92000 + 30000 + 20000 + 10000 = 663060元/日

### 5.6 单位成本

**计算公式**：
```
单位成本 = 总成本 / 纸产量
```

**实例**：
- 总成本：663060元/日
- 纸产量：100吨/日
- 单位成本：663060 / 100 = 6630.6元/吨

## 6. 物料平衡计算

### 6.1 抄造过程物料平衡

**输入**：
- 绝干纸浆：M₁
- 添加剂：M₂
- 填料：M₃
- 水：V₁

**输出**：
- 成品纸：M₄（含水分）
- 白水：M₅（含固形物）
- 损耗：M₆

**平衡方程**：
1. 固形物平衡：M₁ + M₂ + M₃ = M₄×(1-水分%) + M₅×固形物浓度% + M₆
2. 水平衡：V₁ = M₄×水分% + M₅×(1-固形物浓度%)

**实例**：
- 输入：绝干纸浆92吨，添加剂5吨，填料18.4吨，水1500m³
- 输出：成品纸100吨（水分8%），白水含固形物3吨
- 计算：
  - 成品纸绝干质量：100×(1-8%) = 92吨
  - 固形物平衡：92 + 5 + 18.4 = 115.4吨
  - 成品纸绝干质量 + 白水固形物：92 + 3 = 95吨
  - 损耗：115.4 - 95 = 20.4吨
  - 水平衡：1500 = 100×8% + 白水含水量
  - 白水含水量：1500 - 8 = 1492吨
  - 白水总质量：1492 + 3 = 1495吨

## 8. 物料计算的应用

### 8.1 工艺优化

通过物料计算，可以：
- 优化纸浆配比，降低成本
- 调整添加剂用量，改善纸张性能
- 减少物料损耗，提高效率

### 8.2 成本控制

通过物料计算，可以：
- 精确计算各种物料的消耗
- 分析成本构成，找出成本控制的重点
- 预测成本变化，制定成本控制措施

### 8.3 生产计划

通过物料计算，可以：
- 制定合理的生产计划
- 确定物料采购计划
- 优化库存管理

### 8.4 质量控制

通过物料计算，可以：
- 控制各种物料的用量，确保产品质量
- 分析物料对产品质量的影响
- 调整物料配比，改善产品质量
