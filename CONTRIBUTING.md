# 贡献指南

感谢您对制浆造纸行业专业知识库项目的关注和支持！我们欢迎各种形式的贡献，以共同完善这个项目。

## 贡献方式

### 1. 内容纠错与更新
- 发现知识库中的错误或过时信息，请提交Issue或直接修改
- 提供更准确、更详细的工艺描述和计算方法

### 2. 补充专业知识
- 补充新的制浆造纸工艺和技术
- 提供实际生产案例和数据（请进行脱敏处理）
- 分享设备操作经验和最佳实践

### 3. 技术贡献
- 改进OpenClaw skill的配置
- 优化系统提示词
- 完善部署指南

## 贡献流程

1. **Fork 仓库**
2. **创建分支**（建议使用功能或修复相关的分支名称）
3. **修改内容**
4. **提交Pull Request**，并在描述中说明您的更改

## 内容规范

- 保持专业、准确的行业术语
- 提供具体的数据和计算方法
- 遵循现有的文件结构和命名规范
- 确保内容符合行业标准和环保要求

## 审核流程

所有贡献将经过审核，确保内容的准确性和专业性。我们会尽快处理您的Pull Request。

## 联系我们

如果您有任何问题或建议，欢迎通过Issue与我们联系。

再次感谢您的贡献！
